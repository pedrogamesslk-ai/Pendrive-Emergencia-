# Kit de Sobrevivência para PC

## Descrição
Este repositório contém um kit de sobrevivência para manutenção e recuperação de sistemas, incluindo ISOs, ferramentas e utilitários.

## Pré-requisitos
O dispositivo de armazenamento deve estar configurado com:
- Partição GPT
- :contentReference[oaicite:0]{index=0} instalado

## Estrutura do projeto
- ISOs/
- Programas/
- Drivers AMD/
- Drivers INTEL/
- ventoy/
## Como usar
1. Instalar o Ventoy no dispositivo
2. Selecionar GPT durante a instalação
3. Copiar as ISOs para o dispositivo
4. Dar boot pelo pendrive/HD externo
